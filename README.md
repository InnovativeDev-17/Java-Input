ğŸ“‚ Program Description

The program performs the following steps:

Asks the user to enter a lucky number.

Reads the number using nextInt().

Displays the entered lucky number.

Asks the user to enter their full name.

Reads the name using nextLine().

Displays the entered name


After using nextInt(), an extra nextLine() is required to clear the input buffer.
Otherwise, the string input may be skipped.
